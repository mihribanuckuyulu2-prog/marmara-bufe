# Aykut Büfe — Marmara Adası

Marmara Adası'nda hizmet veren Aykut Büfe için hazırlanmış tanıtım web sitesi.

## Müşteri

**Aykut Büfe** — Marmara Adası Merkez'de faaliyet gösteren, tost, hamburger, köfte, döner, kumru, çibörek gibi lezzetler sunan yerel bir büfe.

## Kullanılan Teknolojiler

- HTML5 (semantik etiketler: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`)
- CSS3 (Flexbox, CSS değişkenleri, responsive tasarım)
- JavaScript (mobil menü toggle, scroll reveal animasyonu)
- Web3Forms (iletişim formu)
- Netlify (HTTPS yayın)

## Özellikler

- Responsive tasarım (mobil ≤768px, tablet 769–1024px, masaüstü ≥1025px)
- Semantic HTML5 yapısı
- Mobil menü aç/kapa
- Scroll ile görünürlük animasyonu
- İletişim formu
- Optimize edilmiş görseller
- Favicon
- Open Graph etiketleri

## Kurulum

Projeyi yerelde çalıştırmak için:

```bash
git clone https://github.com/KULLANICI_ADI/aykut-bufe.git
cd aykut-bufe
# index.html'i tarayıcıda aç
```

İletişim formunun çalışması için `index.html` içindeki `YOUR_ACCESS_KEY_HERE` yerine [Web3Forms](https://web3forms.com)'dan alacağınız access key'i yazın.
